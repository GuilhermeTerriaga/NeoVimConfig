# Neovim config

My neovim config!

# Installation

Just clone the repo and place it on .config/nvim/

# Plugins

- Ayu colorscheme
- barbar as buffer bar
- Smart completions with cmp-nvim-lsp, nvim-autopairs, luaSnip and nvim-cmp
- CsvView to handle CSVs
- Nvim DAP to debug
- Git-conflict to handle git conflicts
